Boardgame Webshop


Please note that this application can't be run directly from your browser (well it can, but won't be functioning as intended), because it needs a Node.js tool called JSON-server for the backend function. This lovely tool simulates the work of a server and gives the appropriate responses for the program.

If you haven't downloaded JSON-server yet, you can simply do it by typing "npm install -g json-server" in your command line.

To run the application, download the code from the link below, copy it to a library on your computer, then type "json-server --watch adatok.json" in your command line (you must be in the library to where you copied the code). This starts the server. Now you can run the program by clicking on index.html.

My main goal with the Boardgame Webshop was to create a fully working webshop with as many functions as possible, while also practicing the various HTTP methods (GET, POST, PUT, PATCH, DELETE). Because of that its appearance is rather functional (and it's not responsive). I'm planning to make version 2 with cleaner code (using async functions) and nicer looks.
